# What can we build for ourselves and other Novi Sad residents?

**Goal:** Generate a prioritized list of neighborhood tools that NSRT residents want to build
**Critical Question:** What is the single most useful tool we could build first for Novi Sad residents?

## Context

NSRT (Novi Sad Relational Tech) is a community initiative building neighborhood tools by, with, and for Novi Sad residents. Part of the NSRelaTech GitHub organization. "Relational technology" means tech that strengthens connections between people — not platforms that extract data and attention. Core principles: neighbor-to-neighbor (not citizen-to-government), village-scale (block, building, mesna zajednica), many small interoperable tools (not one monolithic platform), remix existing ideas, community sovereignty over tools and data. Results from this session will feed into the needs-discovery research repository.
