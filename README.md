# Needs Discovery

What do Novi Sad neighbors actually need? This repo collects discovery research, Harmonica session results, and community insights that guide what [NSRelaTech](https://github.com/NSRelaTech) builds.

## Structure

```
sessions/    # Summaries and key takeaways from Harmonica deliberation sessions
findings/    # Synthesized insights, patterns, and recommendations
```

## Context

A recurring theme in the NSRT community is the gap between what tools exist and what neighbors actually need. This repo serves as an R&D hub — capturing what we learn from structured conversations, informal feedback, and direct observation.

Discovery feeds into the tools we build: [nsrt](https://github.com/NSRelaTech/nsrt) (landing page and directory), [dear-neighbors](https://github.com/NSRelaTech/dear-neighbors) (neighborhood dashboard extension), and whatever comes next.

## Contributing

Run a session, talk to neighbors, notice a pattern — write it up and open a PR. No format requirements yet; clarity over polish.
